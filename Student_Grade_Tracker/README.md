# Student Grade Tracker

## Project Description
This is a Java-based Student Grade Tracker developed for the CodeAlpha Java Programming Internship.

## Features
- Add student names and marks
- Calculate average marks
- Find highest and lowest marks
- Display topper
- Assign grades
- PASS/FAIL result
- Save report in StudentReport.txt

## Technologies Used
- Java
- Eclipse IDE
- Scanner
- Arrays
- FileWriter

## How to Run
1. Open the project in Eclipse.
2. Run StudentGradeTracker.java.
3. Enter student details.
4. View the report.
