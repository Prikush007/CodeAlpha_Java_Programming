# CodeAlpha AI ChatBot

## Project Description
This project is a Java-based AI ChatBot developed as part of the CodeAlpha Internship.

## Features
- Java Swing GUI
- Interactive Chat Window
- Multiple Commands
- Date & Time
- Motivational Quotes
- Jokes
- Help Command
- Chat History Saving

## Technologies Used
- Java
- Eclipse IDE
- Java Swing
- AWT
- FileWriter

## How to Run
1. Open the project in Eclipse.
2. Run ChatBotGUI.java.
3. Type your message.
4. Click Send or press Enter.
